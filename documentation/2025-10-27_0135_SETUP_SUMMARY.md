## 📋 Setup Checklist

### 1. Get Resend API Key (5 min)
- [ ] Go to https://resend.com/
- [ ] Sign up / log in
- [ ] Create API key
- [ ] Copy key (starts with `re_`)

### 2. Add Domain to Resend (10 min)
- [ ] In Resend, go to "Domains"
- [ ] Add `admin.stafflyhq.ai`
- [ ] Copy DNS records shown

### 3. Add DNS Records (10 min)
Add these to your DNS provider for `stafflyhq.ai`:

```
Type: TXT
Name: admin.stafflyhq.ai
Value: [from Resend]

Type: MX  
Name: admin.stafflyhq.ai
Value: [from Resend]
Priority: 10

Type: TXT
Name: resend._domainkey.admin.stafflyhq.ai
Value: [from Resend - DKIM]
```

### 4. Wait for Verification (5-30 min)
- [ ] Wait for DNS propagation
- [ ] Click "Verify" in Resend
- [ ] See green checkmark ✅

### 5. Set API Key in Supabase (2 min)
```bash
cd /Users/jeladiaz/Documents/StafflyFolder/dealdashai
npx supabase secrets set RESEND_API_KEY=re_your_key_here
```

### 6. Test (5 min)
- [ ] Submit an EOD report
- [ ] Check miguel@migueldiaz.ca
- [ ] Check client email
- [ ] Verify email format

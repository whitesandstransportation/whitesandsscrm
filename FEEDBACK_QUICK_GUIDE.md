# Feedback System - Quick Start Guide

## 🎯 For DAR Users

### How to Submit Feedback

1. **Open DAR Portal** and log in
2. **Click "Feedback"** in the left sidebar (below Settings)
3. **Fill out the form**:
   - **Subject**: Brief description (e.g., "Bug: Timer not stopping")
   - **Message**: Detailed explanation of your feedback
4. **Add Images** (optional):
   - **Method 1**: Click "Choose File" button
   - **Method 2**: Copy an image and paste (Ctrl+V or Cmd+V) directly into the message field
5. **Click "Submit Feedback"**
6. ✅ Done! You'll see a success message

### Tips
- Be specific in your subject line
- Include steps to reproduce if reporting a bug
- Screenshots are very helpful!
- You can paste multiple images

---

## 👨‍💼 For Admins

### How to View and Respond to Feedback

1. **Open Admin Dashboard**
2. **Click the "Feedback" tab** (next to DAR Live)
3. **View the feedback table** showing:
   - User name and email
   - Subject
   - Status badge (color-coded)
   - Submission date
4. **Click "View Details"** on any feedback to see:
   - Full message
   - Attached images (click to open full size)
   - Admin response area
5. **Update the status**:
   - New → In Progress (when you start working on it)
   - In Progress → Resolved (when fixed)
   - Resolved → Closed (when confirmed complete)
6. **Write a response** in the text area
7. **Click "Save Response"**

### Status Meanings
- 🔵 **New**: Just submitted, needs attention
- 🟡 **In Progress**: Currently being worked on
- 🟢 **Resolved**: Issue has been fixed
- ⚫ **Closed**: Feedback addressed and archived

## 📸 Image Features

### Supported Methods
1. **File Upload**: Click "Choose File" → Select image
2. **Copy & Paste**: 
   - Take a screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
   - Paste directly into the message field (Ctrl+V / Cmd+V)
3. **Remove Images**: Hover over image → Click X button

### Supported Formats
- PNG, JPG, JPEG, GIF, WebP
- Any standard image format

---

## 🔔 What Happens After Submission?

### For Users:
- Your feedback is immediately saved
- Admins can see it in their dashboard
- You'll receive updates when admins respond (check back in Feedback tab)

### For Admins:
- New feedback appears at the top of the list
- You can track all feedback history
- Responses are saved and visible to users

## 💡 Best Practices

### When Submitting Feedback:
- ✅ Use clear, descriptive subjects
- ✅ Include what you expected vs. what happened
- ✅ Attach screenshots when relevant
- ✅ Mention if it's urgent
- ❌ Don't submit duplicate feedback

### When Responding (Admins):
- ✅ Update status promptly
- ✅ Provide clear responses
- ✅ Include timeline if applicable
- ✅ Ask for more info if needed
- ❌ Don't leave feedback in "New" status too long

---

## 🆘 Troubleshooting

### Can't see Feedback tab?
- Make sure you're logged in
- Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)

### Image won't upload?
- Check file size (should be reasonable)
- Try a different image format
- Check your internet connection

### Feedback not submitting?
- Make sure both Subject and Message are filled
- Check for error messages
- Try refreshing and submitting again
